# CalculadoraIMC

Projeto teste da Digital Innovation One com o intuito de aplicar os conteúdos abordados no **Bootcamp everis Kotlin Developer**.

Este aplicativo faz IMC(Índice de Massa Corporal) de uma pessoa para a sua análise feita em uma tabela de referência de seu peso ideal.
